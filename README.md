# Coffee Shop Dashboard

## Project Overview
This project is an interactive Coffee Shop Sales Dashboard created in Microsoft Excel. It helps analyze sales performance, customer behavior, and product trends using charts, pivot tables, and slicers.

## Features
- Total Sales Analysis
- Monthly Sales Trends
- Top Selling Products
- Category-wise Sales
- Store-wise Performance
- Interactive Filters (Slicers)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dataset
Coffee Shop Sales Dataset
